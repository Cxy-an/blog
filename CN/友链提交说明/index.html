目前友链申请状态: 开放

若想通过本页面申请本站的友链，需满足以下硬性条件:

条件

  1. 目前只支持个人博客进行申请。商业网站恕不接受友链申请。
  2. 网站需要有独立的域名，不支持 IP 地址和第三方二级域名 (即允许你的独立域名下的二级域名)。
  3. 博客需要保证持续更新，且博客中必须保证大部分内容均为原创内容。
  4. 网站支持 HTTPS。
  5. 网站如被盗，请立即联系站长删除，否则将会加入黑名单

注意事项

  1. 网站名称 / 描述 / Logo 均需要符合相关法规，否则不予通过。
  2. 如果选择不缓存你的网站 Logo，则友链图标会直接从你提供的 Logo URL 加载，好处是你可以随时通过更新 Logo URL 指向的图片来更新友链图标。(但是你不能通过这种形式把图标换成违规图片，一经发现直接将你网站的友链删除)
  3. 申请结果博主会在本页中你的申请中进行回复，请注意留意。
